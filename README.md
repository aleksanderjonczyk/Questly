# Questly

## Questly is an app where you could set recurring tasks called Rituals as well as one-time tasks called Quests.

### MVP

- Create Quests and Rituals
- Complete them to earn XP
- View what's waiting for you today and in future
- Stats summary

### Stack

- Vite + React + TypeScript
- React Router
- json-server ("backend")
- SCSS for styles

### Get started

- install dependencies
- run app
- run json-server
